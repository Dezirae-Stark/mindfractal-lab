# Image Embedding Reference

This file contains ready-to-use Markdown snippets for embedding MindFractal Lab visualizations.

---

## Phase Portrait

**Markdown:**
```markdown
![Phase Portrait](docs/images/phase_portrait.png)
```

**HTML (with caption):**
```html
<figure>
  <img src="docs/images/phase_portrait.png" alt="Phase Portrait" width="600">
  <figcaption><em>Figure 1: Phase portrait showing trajectory evolution in state space.</em></figcaption>
</figure>
```

**Centered (GitHub):**
```html
<p align="center">
  <img src="docs/images/phase_portrait.png" alt="Phase Portrait" width="600">
</p>
```

## LaTeX Integration (for papers)

```latex
\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.8\textwidth]{docs/images/phase_portrait.png}
    \caption{Phase portrait of the fractal dynamics model showing chaotic trajectory.}
    \label{fig:phase_portrait}
\end{figure}
```

---

**Note:** All images are generated by scripts in the `examples/` directory:
- `phase_portrait_script.py` → `phase_portrait.png`
- `basin_script.py` → `basin.png`
- `lyapunov_param_space_script.py` → `lyapunov_param_space.png`
- `attractor_3d_script.py` → `attractor_3d.png`
- `trajectory_gif_script.py` → `trajectory_on_basin.gif`
