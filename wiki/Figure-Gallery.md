# Figure Gallery

Visual examples generated by MindFractal Lab.

## Phase Portrait

![Phase Portrait](../docs/images/phase_portrait.png)

**Description:** Trajectory evolution in 2D state space showing the path from initial condition to attractor.

**Script:** `examples/phase_portrait_script.py`

**Parameters:**
- `c = [0.5, 0.3]`
- `x0 = [0.1, 0.1]`
- `n_steps = 5000`

---

## Basin of Attraction

![Basin of Attraction](../docs/images/basin.png)

**Description:** Color-coded regions showing which initial conditions converge to which attractor. Fractal boundary structure indicates sensitivity to initial conditions.

**Script:** `examples/basin_script.py`

**Parameters:**
- `c = [0.3, 0.2]`
- `resolution = 300×300`
- `x_range = [-3, 3]`
- `y_range = [-3, 3]`
- `n_steps = 500`

---

## Lyapunov Parameter Space

![Lyapunov Parameter Space](../docs/images/lyapunov_param_space.png)

**Description:** Heatmap showing largest Lyapunov exponent across parameter space. Red regions indicate chaos (λ > 0), blue regions indicate stability (λ < 0). Black contour marks the chaos boundary (λ = 0).

**Script:** `examples/lyapunov_param_space_script.py`

**Parameters:**
- `c1_range = [-2, 2]`
- `c2_range = [-2, 2]`
- `resolution = 150×150`

---

## 3D Attractor

![3D Attractor](../docs/images/attractor_3d.png)

**Parameters:**

## Animated Trajectory

![Trajectory Animation](../docs/images/trajectory_on_basin.gif)

**Description:** Animated visualization showing a trajectory evolving on the basin of attraction background. The white dot represents the current state, with a fading trail showing recent history.

**Script:** `examples/trajectory_gif_script.py`

**Parameters:**
- `c = [0.3, 0.2]`
- `x0 = [-2.0, 1.5]`
- `n_frames = 200`
- `fps = 20`

## Generating Figures

Run all figure generation scripts:

```bash
cd mindfractal-lab

# Static figures
python examples/phase_portrait_script.py
python examples/basin_script.py
python examples/lyapunov_param_space_script.py
python examples/attractor_3d_script.py

# Animated GIF
python examples/trajectory_gif_script.py
```

All outputs are saved to `docs/images/`.
